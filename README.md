# Portfolieo-Project-

# Modern AI Engineer & Full Stack Developer Portfolio

A sleek, responsive, and highly interactive personal portfolio website showcasing full-stack web applications and artificial intelligence integrations. Built using a performance-first aesthetic, featuring custom interactive UI components and smooth scroll animations[cite: 1, 3].

## 🚀 Key Features

*   **Interactive Custom Cursor:** An elegant mouse-tracking `cursor-dot` paired with a smoothly lagging `cursor-trail` that reacts upon hovering over interactive elements[cite: 1, 3].
*   **Parallax Card Effects:** Dynamic mouse-tracking translation applied to the hero visual, offering a modern 3D shifting look[cite: 1, 3].
*   **Scroll Reveal Animation:** Seamless element fade-ins and slide-ups powered by the `IntersectionObserver` API[cite: 1, 3].
*   **Sophisticated Theme:** A crisp, high-contrast palette built on a tailored selection of deep inks, vibrant blues, and electric violets[cite: 1, 4].

---

## 🛠️ Tech Stack & Directory Breakdown

The project repository is consolidated from standard modular front-end layers[cite: 1, 3, 4]:

*   **HTML5 (`index.html`):** Clean, semantic layout defining responsive architecture grids for optimal scannability.
*   **CSS3 (`styles.css`):** Utilizing centralized CSS Custom Properties (`:root`), flex/grid structures, and hardware-accelerated animations (`@keyframes`)[cite: 4].
*   **JavaScript (`script.js`):** Lightweight, zero-dependency performance logic driving standard DOM events, custom micro-interactions, and tracking[cite: 3].
