# Modern AI Engineer & Full Stack Developer Portfolio

A sleek, responsive, and highly interactive personal portfolio website showcasing full-stack web applications and artificial intelligence integrations. Built using a performance-first aesthetic, featuring custom interactive UI components and smooth scroll animations.

## 🚀 Key Features

* **Interactive Custom Cursor:** An elegant mouse-tracking `cursor-dot` paired with a smoothly lagging `cursor-trail` that reacts upon hovering over interactive elements.
* **Parallax Card Effects:** Dynamic mouse-tracking translation applied to the hero visual, offering a modern 3D shifting look.
* **Scroll Reveal Animation:** Seamless element fade-ins and slide-ups powered by the `IntersectionObserver` API.
* **Sophisticated Theme:** A crisp, high-contrast palette built on a tailored selection of deep inks, vibrant blues, and electric violets.

## 🛠️ Tech Stack & Directory Breakdown

The project repository is consolidated from standard modular front-end layers:

* **HTML5 (`portfolieo.html`):** Clean, semantic layout defining responsive architecture grids for optimal scannability.
* **CSS3 (`style.css`):** Utilizing centralized CSS Custom Properties (`:root`), flex/grid structures, glassmorphism overlays, and hardware-accelerated animations (`@keyframes`).
* **JavaScript (`script.js`):** Lightweight, zero-dependency performance logic driving standard DOM events, custom cursor micro-interactions, and parallax tracking.
